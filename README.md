PF - Prevented Fraction
==


### To install or update PF:

From **within R**

```
require(devtools)
install_github("ABS-dev/PF")
```


### Package Vignette:

Read online at https://github.com/ABS-dev/PF/blob/master/inst/doc/PFvignette.pdf

From **within R**

```
vignette("PFvignette", package = "PF")
```
