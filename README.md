PF - Prevented Fraction
==


### To install or update PF:

From **within R**

1. Installing last release v9.5.1

```
require(devtools)
install_github("ABS-dev/PF", ref = "v9.5.1")
```

2. Installing work-in-progress towards v9.5.3

```
require(devtools)
install_github("ABS-dev/PF")
```
